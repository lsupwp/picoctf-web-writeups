# 🛡️ PicoCTF Writeups – Web Exploitation

## 📌 About
This repository contains a collection of writeups for PicoCTF challenges, focused on **Web Exploitation**.

It is intended as a learning resource for:
- Analyzing CTF challenges
- Identifying and understanding vulnerabilities
- Learning exploitation techniques
- Writing structured and clear security writeups

---

## 🎯 Objective
- Improve cybersecurity and web security skills  
- Understand real-world attack techniques  
- Learn how vulnerabilities are exploited and mitigated  

---

## 📂 Contents
Example topics included in this repository:

- NO FA (2FA Bypass / Session Vulnerability)
- Authentication Bypass
- Information Disclosure
- Session Exploitation

## 🧠 Skills Covered
- Web Application Analysis  
- Flask Session Handling  
- Hash Cracking (SHA256 Lookup)  
- Cookie / Session Exploitation  
- Authentication & Authorization Bypass
