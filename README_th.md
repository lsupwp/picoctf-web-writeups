# 🛡️ PicoCTF Writeups – Web Exploitation

## 📌 About
Writeup ชุดนี้จัดทำขึ้นเพื่อใช้เป็นแหล่งเรียนรู้เกี่ยวกับ:

- กระบวนการวิเคราะห์โจทย์ CTF
- การค้นหาและวิเคราะห์ช่องโหว่ (Vulnerability Analysis)
- เทคนิคการโจมตีระบบ (Exploitation)
- การทำ Writeup อย่างเป็นระบบ

เนื้อหาครอบคลุมโจทย์ประเภท **Web Exploitation** จาก PicoCTF

---

## 🎯 Objective
- ฝึกทักษะด้าน Cybersecurity และ Web Security
- เข้าใจแนวคิดของการโจมตีระบบจริง
- เรียนรู้การป้องกันช่องโหว่จากมุมมองของผู้โจมตี

---

## 📂 Contents
ตัวอย่างหัวข้อภายใน repository:

- NO FA (2FA Bypass / Session Vulnerability)
- Authentication Bypass
- Information Disclosure
- Session Exploitation

## 🧠 Skills Covered

- Flask Session Handling
- Hash Cracking (SHA256 Lookup)
